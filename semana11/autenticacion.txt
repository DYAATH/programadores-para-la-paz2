El login permite identificar a un usuario antes de darle acceso a las funciones de una aplicación. Gracias a la autenticación se puede controlar quién tiene permiso para consultar o modificar determinada información. Algunas rutas deben estar protegidas porque contienen datos privados o funciones administrativas. No se deben exponer públicamente contraseñas, documentos de identidad, direcciones, teléfonos, tokens, claves de API ni otra información sensible. Esto ayuda a proteger la privacidad y la seguridad de los usuarios.
